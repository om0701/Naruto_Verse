# NarutoVerse – Explore the Shinobi World

NarutoVerse is a responsive web application built using HTML, CSS, and JavaScript powered by the Dattebayo API.  
It allows users to explore the Naruto universe, browse characters, clans, and villages, and view detailed information with smooth transitions and real-time search.

---

## Live Demo
https://om0701.github.io/Naruto_Verse/
---

## Features

- Asynchronous data fetching using Fetch API and async/await
- Dynamic pagination for characters, clans, and villages
- Real-time live search with debounce functionality
- Detailed modal view for characters, clans, and villages
- Smooth scrolling and content transitions
- Loading animation and skeleton placeholders
- Error and 404 handling with Genjutsu-style message
- Dark and Light theme toggle with localStorage
- Cached API responses for faster reloads
- SEO optimized with meta tags, Open Graph, and Twitter previews

## Functionalities Implemented

1. Fetching of data asynchronously from the Dattebayo API.
2. Pagination implemented for characters, clans, and villages.
3. Real-time live search that updates dynamically as the user types.
4. Smooth scroll animation for the “Browse Characters” button.
5. Skeleton loading animation for smooth data transition.
6. Error handling for API timeouts and server errors.
7. Modal displaying detailed information including jutsu, rank, and family.
8. Theme toggle for Dark and Light modes with persistent storage.
9. Optimized meta tags for SEO and social sharing preview.
10. Custom favicon added for brand identity.
